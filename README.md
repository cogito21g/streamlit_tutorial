## Setting

- 가상환경설정
```bash
conda create -n streamlit python=3.10
conda activate streamlit
```

- 패키지 설치
```bash
pip install streamlit
```

- 실행
```bash
streamlit run app.py
# 포트 및 IP 주소 설정
streamlit run main.py --server.port 30081 --browser.serverAddress 0.0.0.0
```
